Необходимо реализовать алгоритм для взвешенного вершинного покрытия графа, указанный на последнем слайде здесь: https://goo.gl/UPeSVK
Реализация должна быть выполнена на C++ (компилироваться без проблем в gcc), Java 8 либо Python 3.
Программа должна читать входные данные со стандартного потока ввода в формате:
<число вершин> <число рёбер>
<весвершины1>
<весвершины2>
…
<началоребра1> <конецребра1>
<началоребра2> <конецребра2>
…

В первой строке подаётся через пробел два целых числа, больших десяти, — количество вершин N и рёбер M графа. 
Далее в N последовательных строках подаются неотрицательные целые веса вершин графа. Далее в M строках подаются через пробел номера концов каждого ребра. Номера вершин графа лежат в диапазоне от 1 до количества вершин. 

На выходе программа должна вывести яерез пробел номера вершин графа, входящие в вершинное покрытие графа, вес которого не более чем вдвое превосходит вес оптимального покрытия.
